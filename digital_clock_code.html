Python Digital-Clock Code

Below is the code for the Python Digital-Clock project.

Note: Before running the code, make sure to install the required libraries by executing the following commands in your command prompt (CMD) or PowerShell:


        pip install tkinter
        pip install ttkthemes

Digital_Clock.py


            from tkinter import *
            from tkinter.ttk import *
            from time import strftime
            
            root = Tk()
            root.title('Clock')
            
            
            def time():
                string = strftime('%I:%M:%S %p')
                # To make the clock format 24 hrs. Simply change the strftime('%I:%M:%S %p') to strftime('%H:%M:%S %p').
                lbl.config(text=string)
                lbl.after(1000, time)  # It will update the time after every (1000 milliseconds i.e. after 1 second)
            
            
            lbl = Label(root, font=('calibri', 40, 'bold'), background='purple', foreground='white')
            
            lbl.pack(anchor='center')
            time()
            
            mainloop()
